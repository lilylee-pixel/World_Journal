Lily's Technology Word Journal

Term 1: Technology

  • My Definition: The application of knowledge, skills, tools, and techniques to solve problems, create new possibilities, or improve the way tasks are performed. It encompasses everything from simple tools and machines to complex systems like computers, the internet, and artificial intelligence. Technology is both a product of human ingenuity and a driver of innovation, shaping how we interact with the world and each other.

  • Synonyms: Tools, Equipment, Machinery, Invention, Media, Innovation, Devices, Applied Science, Gadgets

  • Antonyms: Natural Processes, Labor, Tradition, Primitive Methods, Handcraft

    Term 2: Computer

  • My Definition: a machine that processes and stores information. It follows instructions from programs to perform tasks like calculations, playing games, or connecting to the internet. Computers have physical parts (hardware) and programs (software) that work together to get things done.

  • Synonyms: PC (Personal Computer), Laptop, Desktop, Workstation, Server, Processor, Terminal, Mainframe, Device, System, Tech.

  • Antonyms: Manual Tools, Analog Tools, Non-electronic Devices, Typewriter, Books, Non-digital Equipment.

    Term 6: Markup Language

    • My Definition: markup language, standard text-encoding system consisting of a set of symbols inserted in a text document to control its structure, formatting, or the relationship between its parts.

    • Synonyms: Tag-based language, nomenclature / terminology, document schema

    • Antonyms: Plain text, programing language, binary file, non-semantic HTML

    Term 10: Algorithm

  • My Definition: a set of mathematical instructions or rules that, especially if given to a computer, will help to calculate an answer to a problem

  • Synonyms: Procedure, process, method, methodology, formula, rule, routine, subroutine, code, script

  • Antonyms: Heuristic, chaos, disorder, randomness, arbitrariness, inaction, idleness

  • Source

    • Term 11: Sequence

  • My Definition: Sequence means putting your instructions in the right order so a computer can follow them correctly.

  • Synonyms: succession, course, series, order, chain, cycle, arrangement

  • Antonyms: disorder, chaos, randomness, set(unordered), jumble, medley

    Term 12: Selections

  • My Definition: code (statements) that is executed only when a certain condition is satisfied.

  • Synonyms: conditionals, decision structures, branching, flow control, boolean logic

  • Antonyms: sequencing, iteration(loops), unconditional branching, rejection, elimination
